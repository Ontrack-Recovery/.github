## Quick Start with Ontrack Recovery

1. **Get Ontrack Recovery** — Grab the portable `.exe` (no installation required).
2. **Launch** — Double-click `Ontrack_Recovery.exe`. The main window opens in under 2 seconds.
3. **Select your disk** — Choose the drive, USB stick, or memory card.
4. **Start quick scan** — Click “Quick Scan”. **Ontrack Recovery** shows deleted files in seconds.
5. **Run deep scan if needed** — For formatted or damaged partitions, click “Deep Scan”. **Ontrack Recovery** analyzes every sector.
6. **Preview and recover** — Select any found file, preview content (images, docs), then click “Restore”.
7. **Work normally** — **Ontrack Recovery** activates only when you need data restoration or disk diagnostics.

## Understanding Ontrack Recovery Core Components

**Ontrack Recovery** is not just another undelete tool. It combines several essential recovery engines:

- **Quick Scan** — Recovers recently deleted files from NTFS, FAT32, exFAT, and ReFS.
- **Deep Scan (signature-based)** — Carves files by headers (JPEG, PDF, DOCX, ZIP, MP4) even without file system metadata.
- **Partition repair module** — Restores lost or overwritten partition tables.
- **Disk hex viewer** — Manually inspect raw bytes on damaged drives.
- **Health diagnostics** — Reads S.M.A.R.T. attributes and surface scan results.
- **Session manager** — Save and resume long deep scans without restarting.
- **Portable recovery logs** — Export scan results to CSV or HTML.

All these work together seamlessly in **Ontrack Recovery**. You never need to switch between separate recovery applications.
